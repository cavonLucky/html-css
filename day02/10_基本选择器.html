通过 CSS 选择器，可以选中页面中的指定元素，从而为这些元素设置样式。
    CSS 中为我们提供了大量的选择器，但是在项目中并不是所有的选择器都很常用

    基本选择器：
      元素选择器：
        - 根据标签名选中多个元素
        - 例子：p{}  div{}  h1{}

      id 选择器：
        - 根据元素的 id 属性选中一个元素
        - 语法：#id{}
        - 列子：#p1{} #box{} #head{}

      类选择器：常用
        - 根据元素的 class 属性选中元素
        - 语法：.class{}
        - - 列子：.p1{} .box{} .head{}

          class 属性和 id 类似，每一个元素都可以指定，用来为元素进行分类，
          和 di 不同，可以为多个元素指定相同的 class ，拥有同一个 class 的元素可以说它们是用一类元素，
          也可以同时为一个元素指定多个 class

      通配选择器：
        - 选择页面中所有的元素
        - *{}
