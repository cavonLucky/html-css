    伪元素：
      - 伪元素是表示的页面中特殊的位置
      - 伪元素是使用 :: 开头
      ::before
        - 元素的开始位置（开始标签之后）
      ::after
        - 元素的结束位置（结束标签之前）
      ::first-letter
        - 首字母
      ::first-line
        第一行
      ::selection
        - 选中的文字的样式

    通过 before、after 可以选中元素开始或结束的位置，从而为其添加内容。
    注意：这里的内容是通过 CSS 添加的，不算网页中的正式内容（比如不能复制）。
      1. 通过它可以添加一些统一的符号
      2. 也可以在特殊场景下通过它们来调整一下页面的样式
