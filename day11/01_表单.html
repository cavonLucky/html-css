    使用input type 为 password来创建密码框，密码框中的内容不会以明文显示，避免密码被偷看

    默认情况下，表单中数据会通过 url 地址来发送，URL地址中?后的内容被称为查询字符串（query string）
      ?username=admin&password=123456
      查询字符串是一个一个的名值对结构，一个数据名对应一个值，多个名值对之间使用&隔开，
      数据发送给服务器后，服务器可以根据数据名获取对应的值

      单选框
        - 使用 input type=radio 来创建一个单选框
            单选框是通过 name 属性来分组的，相同 name 属性的为一组
            像这种选择框，不需要用户填写内容，还必须为表单项指定value，value最终会成为提交给服务器的值
      多选框
        - 使用 input type=checkbox 来创建多选框

      下拉列表
        - 使用 select 来创建下拉列表
        - 添加 multiple 属性后可以将下拉列表设置为多选的下拉列表
